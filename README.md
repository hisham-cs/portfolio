# Hisham Al-Malki Portfolio

Personal portfolio website showcasing my background, selected projects, skills, education, and contact links in a clean modern interface.

**Live Website:** https://hisham-cs.github.io/portfolio/

## Features

- Responsive personal portfolio
- Modern dark UI with purple accent
- About, skills, projects, education, and contact sections
- External links for GitHub, LinkedIn, live demos, and project repositories

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## Getting Started

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Deployment

This portfolio is deployed using GitHub Pages. To deploy manually:

```bash
npm run deploy
```
