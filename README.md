# Hisham Al-Malki Portfolio

This is my personal portfolio website showcasing my practical projects in AI, Machine Learning, Computer Vision, and Data Analysis. It highlights my focus on building applied AI solutions and data-driven systems.

## Features

- Responsive personal portfolio
- Dark AI-inspired UI
- Hero section with profile summary
- About section
- Skills section
- Projects section
- Education and certificates section
- Contact section
- Project links and live demo links where available

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## Website Sections

- **Hero**: Profile summary, focus areas, and quick links.
- **About**: A brief introduction to my background, interests, and practical approach to AI and data.
- **Skills**: Core skills categorized by AI/ML, Data & BI, and Tools & Platforms.
- **Projects**: A detailed showcase of my applied AI and data analytics projects.
- **Education & Certificates**: My academic background and recent certifications.
- **Contact**: Get in touch via email or professional networks.

## Getting Started

To run the project locally, run the following commands:

```bash
npm install
npm run dev
```

To build the project for production:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```

## Deployment

GitHub Pages deployment is configured using the `gh-pages` package. 

To deploy the site manually to the `gh-pages` branch, run:

```bash
npm run deploy
```

*Note: The `base` property in `vite.config.js` should match your repository name for GitHub Pages deployment to work correctly.*
